# DRG Info Demo

A simple static demo page for testing deployment on Cloudflare Pages.

## Files
- index.html — main page content
- styles.css — styling for the demo

## Local preview
Open the page in a browser at:
- http://localhost:8000/

## Cloudflare Pages deployment
1. Create a new Cloudflare Pages project.
2. Connect this repository.
3. Set the build output directory to the project root.
4. Deploy.

This page is intentionally simple so it can be used for initial validation before a full production deployment.
